# Customer Satisfaction — Factor Analysis

## Descripción del Proyecto

Este proyecto realiza un Análisis Factorial sobre un conjunto de datos de satisfacción del cliente con el objetivo de:

* Explorar la estructura subyacente de las variables.
* Determinar el número óptimo de factores mediante autovalores y Scree Plot.
* Evaluar la idoneidad del dataset con KMO y prueba de Bartlett.
* Extraer factores con método principal y rotación Varimax.
* Interpretar factores y analizar su relación con outcomes del negocio.
* Generar visualizaciones y conclusiones ejecutivas.

---

## Estructura del Notebook

1. EDA inicial del dataset
2. Pruebas de idoneidad (KMO, Bartlett)
3. Selección del número de factores (autovalores, varianza, scree plot)
4. Extracción y rotación de factores (Varimax)
5. Construcción de puntajes factoriales
6. Visualizaciones
7. Conclusiones y recomendaciones
